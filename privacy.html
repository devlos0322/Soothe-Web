1. 수집하는 정보

로컬 사용 데이터

  • 계획, 기록, 카테고리, 설정 등 앱 사용 중 생성한 데이터
  • 기본적으로 기기 내 저장소에 저장

Google 연동 사용 시

  • Google 계정 정보(계정 식별자, 이메일, 이름)
  • Google Drive 백업 데이터
  • Google Calendar 동기화 데이터(이벤트 제목, 시간, 설명)

2. 정보 이용 목적

  • 앱 핵심 기능 제공(계획/기록/인사이트)
  • 계정 연동 상태 표시 및 인증 유지
  • 백업/복원 및 캘린더 동기화 처리

3. 보관 및 파기

  • 로컬 데이터는 사용자 삭제 전까지 기기에 보관
  • Google 연동 데이터는 Google 정책을 따름
  • 로그아웃/삭제 시 관련 로컬 정보 제거

4. 제3자 제공 및 추적

  • 광고 목적의 데이터 판매 없음
  • 사용자 요청 시 Google OAuth/Drive/Calendar로 기능 수행 데이터 전달 가능
  • 사용자 추적(Tracking) 및 맞춤형 광고 데이터 수집 없음

5. 이용자 권리 및 문의

  • 앱 내 데이터 삭제 가능
  • Google 연동 해제(로그아웃) 가능
  • 문의 채널: 지원 페이지 / GitHub Issues
