# Soothe 개인정보처리방침

[메인 페이지](https://devlos0322.github.io/Soothe-Web/) | [지원 페이지](https://devlos0322.github.io/Soothe-Web/support.html)

최종 업데이트: 2026-03-16

Soothe(이하 "앱")는 사용자 데이터의 최소 수집과 투명한 처리를 원칙으로 합니다.

## 1. 수집하는 정보

앱은 다음 정보를 처리할 수 있습니다.

1) 로컬 사용 데이터
- 계획, 기록, 카테고리, 설정 등 앱 사용 중 생성한 데이터
- 기본적으로 기기 내 저장소에 저장됩니다.

2) Google 연동 사용 시
- Google 계정 정보: 계정 식별자, 이메일, 이름(제공되는 경우)
- Google Drive 백업 데이터: 사용자가 앱에서 생성한 백업 JSON 데이터
- Google Calendar 동기화 데이터: 이벤트 제목, 시간, 설명

## 2. 정보 이용 목적

- 앱 핵심 기능 제공(계획/기록/인사이트)
- 계정 연동 상태 표시 및 인증 유지
- 사용자 요청에 따른 백업/복원 및 캘린더 동기화

## 3. 정보 보관 및 파기

- 로컬 데이터는 사용자가 삭제하기 전까지 기기 내 보관됩니다.
- Google 연동 데이터는 사용자의 Google 계정/서비스 보관 정책을 따릅니다.
- 사용자가 앱에서 로그아웃하거나 데이터를 삭제하면 관련 로컬 정보는 제거됩니다.

## 4. 제3자 제공

앱은 광고 목적의 제3자 데이터 판매를 하지 않습니다.

다만 사용자가 연동 기능을 사용할 때, 기능 수행을 위해 아래 서비스에 데이터가 전달될 수 있습니다.
- Google OAuth (로그인)
- Google Drive API (백업/복원)
- Google Calendar API (이벤트 동기화)

## 5. 추적 및 광고

- 앱은 사용자를 추적(Tracking)하지 않습니다.
- 맞춤형 광고를 위한 데이터 수집을 하지 않습니다.

## 6. 이용자 권리

사용자는 다음을 요청/실행할 수 있습니다.
- 앱 내 데이터 삭제
- Google 계정 연동 해제(로그아웃)
- Google Drive/Calendar에 저장된 연동 데이터 삭제(각 서비스에서 직접 관리 가능)
